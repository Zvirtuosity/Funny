# 从2D图形转出3D建模.
  ![image](https://github.com/Zvirtuosity/01. 从2D图形转出3D建模/1.jpg)   
## AI文件
  Photoshop处理导出路径, 通过反选选中区域, 然后路径->路径,再路径选中工具选中, 导出路径AI文件。  
  ![image](https://github.com/Zvirtuosity/01. 从2D图形转出3D建模/path.jpg)  

## 3ds Max 挤压
  3ds Max导入选AI文件, 然后选中挤压.调整参数,数量参数改一下。
